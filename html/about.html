Welcome to pinksheetmafia.com

This is a free, community-built, open-source resource designed to make NIFE less stressful. Here you’ll find problem generators and solvers—primarily for Navigation, plus Weather and FR&R—as well as (eventually...) curated gouge documents and high-quality practice questions, all submitted and vetted by students who have successfully made it through NIFE.

This project is a work in progress (launched 18AUG25) and is actively being developed. If you spot issues or have ideas for new features, please reach out at drew.Loevinger@gmail.com.

The value of this resource depends on the community:

  • Submit questions and docs you think will help future students.
  • Edit or downvote outdated or incorrect content.
  • If you’re comfortable with code, contribute directly via our open-source GitHub repo.
